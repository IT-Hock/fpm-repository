About Fish Package Manager

FPM is a package manager for the Fish Shell. It is a simple tool to install and manage packages and themes for the Fish Shell. It is written in Go and is available for Linux and macOS.

For more information about FPM, please visit the GitHub Repository.

What is this site?

This site is the official companion site for the FPM package manager. Here you'll find a neatly organized list, with a search function and a details page of all packages and themes that are available in the repository.

Why another package manager for the fish shell?

The fish shell is a great shell, but it lacks a package manager. There are some projects that try to solve this problem, but they are in our opinion not really good. The goal of FPM is to be a simple and easy to use package manager as other package managers like npm, yarn or pip. Because installing packages and themes should not be a hunt for the right files, name for the repository.

How does it look like?

Installing a package is as easy as typing fpm install package-name. FPM will then search for the package on the official repository hosted here.
